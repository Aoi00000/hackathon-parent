# Demo Day 用シナリオ

1. `user1@example.com / user1111` でログインする。
2. 商品一覧で初期20商品が出ていることを見せる。
3. 左サイドバー検索でカテゴリ・価格を絞り込む。
4. 自然言語検索に `参考書 300円 ~ 1500円` と入力し、価格範囲とカテゴリが自動反映されることを見せる。
5. 商品詳細で画像カルーセル、公開コメント、非公開DM、AI質問応答を見せる。
6. 出品画面で複数画像をアップロードし、×で削除できることを見せる。
7. AIで商品説明を生成し、Vertex AIが使えない場合もローカル生成で止まらないことを説明する。
8. 購入手続き、発送通知、受け取り評価、通知ページを見せる。
9. マイページで残高、今月/累計の売上・利用額、出品者評価を見せる。
10. MerRec分析は、`ml/` のstreamingスクリプトで実データを読み込めることを説明する。
