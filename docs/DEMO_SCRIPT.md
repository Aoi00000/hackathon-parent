# Demo Day 用シナリオ

## 0. 最初に説明する一言

Regatear ~ AI Flea Market ~ は、通常のフリマアプリに、自然言語検索、購入前AIチェック、価格交渉アシスタント、AI対話スレッド、販売改善通知、MerRec分析、支払い・通知・評価までを組み込んだ「次世代フリマアプリ」である。

## 1. 商品一覧と自然言語検索

1. `user1@example.com / user1111` でログインする。
2. 商品一覧で初期20商品が表示されることを見せる。
3. サイドバーでカテゴリ、価格、状態などを絞り込む。
4. 自然言語検索に `参考書 300円 ~ 1500円` と入力し、検索条件へ自動変換されることを見せる。
5. おすすめ商品がDB内の商品から安定して表示されることを説明する。

## 2. 商品詳細、購入前AI、価格交渉、コメント/DM

1. 商品詳細を開く。
2. 画像・動画カルーセルを見せる。
3. 購入前AIチェックで、不安点、質問候補、不整合、価格感が出ることを見せる。
4. AIに商品について質問する。
5. 価格交渉アシスタントに希望金額を入力し、購入検討者向けの丁寧な交渉文が出ることを見せる。
6. 出品者アカウントでは、承諾、代替案、お断りの文面が出ることを説明する。
7. 公開コメントと返信スレッドを見せる。
8. 非公開DMは関係者だけが見られることを説明する。
9. 出品者は自分の商品についた公開コメントを削除できることを説明する。

## 3. 出品体験

1. 出品画面を開く。
2. 複数画像と短い動画をアップロードする。
3. ドラッグ&ドロップでメディア順を入れ替える。
4. 必須項目のチェックを見せる。
5. AI商品説明生成を実行する。
6. 出品完了後に通知が届くことを見せる。

## 4. 履歴ページのUX

1. 出品履歴を開き、左にAvailable、右にSOLDが分かれて表示されることを見せる。
2. Availableは古い順で、長く売れ残っている商品を上から確認できることを説明する。
3. SOLDは新しい順で、直近の取引を確認しやすいことを説明する。
4. 購入履歴では、未完了取引と取引完了済みが左右に分かれることを見せる。
5. チェックリストでは、気になる商品の販売中/売り切れを左右に分けて確認できることを見せる。
6. いずれも検索対象であり、商品概要クリックで詳細へ遷移できることを見せる。

## 5. AI対話スレッド

1. ヘッダーからAI対話ページを開く。
2. 「新しい話題」を作る。
3. 例文 `休日の遊びのおすすめない？` を送信する。
4. 回答の最後に「おすすめグッズ」が一般名で出ることを見せる。
5. 別の話題として `家の模様替えをしてみたいんだけどいい案ない？` を送信する。
6. 左側のスレッドリストで話題ごとに履歴が分かれていることを見せる。
7. ページを移動して戻っても履歴が残ることを説明する。

## 6. マイページ、支払い方法、収支グラフ

1. マイページを開く。
2. 残高、今月/累計売上額、今月/累計利用額、出品者評価を見せる。
3. 月別売上額と利用額の棒グラフを見せる。
4. 支払い方法未登録時はチャージできず、登録欄へ誘導されることを説明する。
5. 支払い方法を登録し、通知ページに「支払い方法登録完了」が届くことを見せる。
6. 残高チャージを行い、通知が届くことを見せる。

## 7. 購入、発送、受け取り評価、売上反映

1. 購入者で商品を購入する。
2. 購入時点で購入者の残高が減ることを見せる。
3. この時点では出品者の残高はまだ増えないことを説明する。
4. 出品者でログインし、購入通知から商品へ移動して発送通知を送る。
5. 購入者でログインし、発送通知から受け取り評価を行う。
6. 出品者で再ログインし、受け取り評価後に残高と売上額が増えたことを確認する。

## 8. 販売改善通知

1. 通知ページに `AI販売改善提案` が表示されることを見せる。
2. 7日以上売れ残っている商品に対して、サイズ追記、キーワード、価格調整のヒントが出ることを説明する。
3. MerRec分析の考え方を、売れ残り改善・価格調整・検索キーワード提案へ接続していることを説明する。

## 9. 最後に強調すること

- 必須CRUDだけでなく、取引フロー全体が通る。
- AIは説明生成だけでなく、検索、購入前チェック、商品質問、価格交渉、生活相談、販売改善通知に広がっている。
- 画像だけでなく動画も扱えるため、商品の状態を伝えやすい。
- 外部AIが落ちてもローカルフォールバックでデモが止まらない。
- 支払い、通知、評価、ブロック、収支グラフまである。
